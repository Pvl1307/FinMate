# FinMate

**FinMate** — персональный финансовый ассистент для учёта доходов, расходов и инвестиций.

## О проекте
Проект помогает контролировать финансы, анализировать траты и принимать обоснованные инвестиционные решения.  
MVP ориентирован на быстрый ввод операций, хранение данных и базовую аналитику.

## MVP функционал
- Учёт доходов и расходов
- Категории и шаблоны операций
- Расчёт текущего баланса
- История операций (день / месяц)
- Минимальная аналитика
- Инвестиционный модуль с подсказкой `buy / hold / sell` с объяснением причины
- CLI (английский) и Telegram-бот (кнопки на русском/английском в зависимости от языка, выбранного пользователем)

## Чего нет в MVP
- Интеграций с банками
- Автоматического подтягивания транзакций
- Веб-интерфейса и Android-приложения
- Сложной математической аналитики (SMA/EMA)
